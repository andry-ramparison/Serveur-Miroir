# Serveur-Miroir
## Éxecution du script de création du miroir

```bash
chmod +x /usr/local/bin/update_mirror.sh
sudo /usr/local/bin/update_mirror.sh
```

## Éxecution du script de dépôt du miroir dans un fichier de configuration

```bash
chmod +x /usr/local/bin/deb_mirror.sh
sudo /usr/local/bin/update_mirror.sh
```
